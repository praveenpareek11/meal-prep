# Pav Bhaji

**Type:** weekday-dish  
**Fridge life:** 1 day · **Freezer-safe:** no  
**Morning active time:** 25 min · **Serves:** 1 person, lunch + dinner  
**Uses prepped components:** onion-tomato masala base — 150 g / 4 heaped tbsp; boiled potatoes — 250 g  
**Source:** Vidhya’s Vegetarian Kitchen Pav Bhaji, adapted as fridge-only version  
**Kitchen-tested:** ❌ not yet  

## Ingredients

| Ingredient                   | Quantity                                     |
| ---------------------------- | -------------------------------------------: |
| Boiled potatoes              | 250 g / 3 medium, peeled and mashed          |
| Onion-tomato masala base     | 150 g / 4 heaped tbsp                        |
| Frozen peas-carrot-beans mix | 1 cup                                        |
| Gobi                         | 1 cup small florets, optional                |
| Capsicum                     | ½ cup, chopped, optional, only if bought that week |
| Oil                          | 2 tsp                                        |
| Butter                       | 1 tsp + extra only while serving             |
| Ginger-garlic paste          | ½ tsp, optional                              |
| Pav bhaji masala             | 1½ tsp                                       |
| Red chilli powder            | ½ tsp                                        |
| Haldi                        | ¼ tsp                                        |
| Salt                         | ¾ tsp or to taste                            |
| Water                        | ¾ to 1 cup                                   |
| Lemon juice                  | 1-2 tsp                                      |
| Onion                        | 2 tbsp, finely chopped, optional garnish     |
| Coriander                    | 1 tbsp, optional                             |

## Method
1. Heat oil and 1 tsp butter in kadai.
2. Add ginger-garlic paste and sauté 30 sec.
3. Add onion-tomato masala base, pav bhaji masala, red chilli powder, haldi, and salt. Cook 3-4 min.
4. Add frozen veg, gobi if using, capsicum if using, and ¾ cup water.
5. Cover and cook 8-10 min till vegetables are soft.
6. Add mashed boiled potatoes.
7. Mash everything together with a potato masher. Add more water if too thick.
8. Simmer 5-7 min till bhaji becomes thick and uniform.
9. Finish with lemon juice, coriander, and a small butter touch only while serving.

## Portion & Store
- Split into 2 portions: lunch + dinner.
- Cool 30-40 min, then refrigerate the dinner portion.
- Fridge-only because this uses potato. Eat within the same day; max next-day lunch if smell and texture are perfect.
- Never freeze pav bhaji in this system.

## What it becomes / variations
- Eat with 3 rotis per meal, dosa, or fresh pav; do not mix into stored rice.
