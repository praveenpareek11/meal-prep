# Onion Green Chilli Poha — 1 person lunch

**Type:** weekday-dish  
**Fridge life:** best eaten fresh; can refrigerate 6-8 hours if needed · **Freezer-safe:** No  
**Active time:** 15-18 min (morning) · **Makes:** 1 lunch portion  
**Uses prepped components:** none  
**Source:** own  
**Kitchen-tested:** ✅ confirmed as of 2026-07-11  

## Ingredients

| Ingredient      | Quantity                        |
| --------------- | ------------------------------: |
| Poha            | 1 cup / 80-90 g, thick poha     |
| Onion           | 1 small, finely chopped         |
| Green chilli    | 1, chopped                      |
| Peanuts         | 1.5 tbsp, optional but recommended |
| Oil             | 1.5 tbsp                        |
| Mustard seeds   | ½ tsp                           |
| Jeera           | ¼ tsp, optional                 |
| Curry leaves    | 6-8, optional                   |
| Hing            | 1 pinch                         |
| Haldi           | ¼ tsp                           |
| Salt            | ¾ tsp, adjust                   |
| Sugar           | ½ tsp, optional                 |
| Lemon juice     | 1-1.5 tsp                       |
| Fresh coriander | optional                        |
| Water           | only for rinsing/sprinkling     |

## Process

1. Put poha in a sieve. Rinse gently under running water for **5-7 seconds only**.
2. Keep aside for **8-10 min** to soften. Fluff with fingers. It should be soft but not mushy.
3. Heat oil in kadai.
4. Add peanuts. Fry on low-medium till slightly crisp. Remove and keep aside, or keep in kadai.
5. Add mustard seeds. Let crackle.
6. Add jeera, hing, curry leaves, green chilli.
7. Add chopped onion + pinch of salt. Cook **3-4 min** till onion turns soft, not fully brown.
8. Add haldi, mix for 10 seconds.
9. Add softened poha, salt, sugar. Mix gently.
10. Sprinkle **1-2 tbsp water**, cover, cook on low flame for **2-3 min**.
11. Switch off. Add lemon juice and peanuts. Mix gently.
12. Rest covered for **2 min**, then eat.

## Notes

- Use **thick poha**, not thin poha.
- Since this skips potato/tomato/veggies, onion + peanuts are important for body and taste.
- For a fuller lunch, have with **curd** or **lehsun chutney/achaar** on the side — the pantry system already keeps poha, curd, achaar, and lehsun chutney stocked.
