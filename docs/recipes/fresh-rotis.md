# Fresh Rotis — 1 person, lunch + dinner

**Type:** weekday-dish  
**Fridge life:** cooked rotis 2-3 days, but best same day · **Freezer-safe:** No  
**Active time:** 25-35 min (morning) · **Makes:** 6 rotis = 3 lunch + 3 dinner  
**Uses prepped components:** none  
**Source:** own  
**Kitchen-tested:** ✅ Yes  

*Storage rule: do **not** store dough; store cooked rotis only, as the plan notes for batch rotis.*

## Ingredients

| Ingredient           | Quantity                  |
| -------------------- | ------------------------: |
| Atta                 | 220 g                     |
| Water                | 140-150 ml, add gradually |
| Salt                 | ¼ tsp optional            |
| Oil or ghee in dough | 1 tsp optional            |
| Ghee for smear       | ½-1 tsp total optional    |

## Process

1. Add **220 g atta + salt** in a bowl/parat.
2. Add **140 ml water gradually** and mix. Add 1-2 tsp extra only if dry.
3. Knead 5-6 min until smooth-medium soft, not sticky.
4. Apply **½ tsp oil/ghee** on top, cover, rest **10-15 min**.
5. Divide into **6 equal balls**.
6. Roll each roti with dry atta.
7. Heat tawa well. Cook roti:

    - Side 1: 20-30 sec, light spots
    - Side 2: 30-40 sec
    - Flip again and puff/press gently

8. Lightly smear ghee if you want softer rotis.

## Storage for lunch + dinner

- **Lunch 3 rotis:** keep wrapped in clean cotton cloth inside a casserole/covered box.
- **Dinner 3 rotis:** cool fully for 20-30 min, wrap in cloth/foil, keep in airtight box, refrigerate.
- Reheat dinner rotis on hot tawa **20-30 sec per side**. Sprinkle a few drops of water only if dry.
