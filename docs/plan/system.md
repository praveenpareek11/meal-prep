# How the System Works

*Solo vegetarian · Hyderabad monsoon · work 12 PM-9 PM · one morning cook covers lunch + dinner.*

## The two-layer model

**Layer 1 — Weekend prep (one session, 1.5-2 hrs):** make *components*, not meals — masala base, plain-boiled legumes and dal, boiled potatoes, chopped veggie boxes, curd, sprouts, batch rotis/theplas, sambar-style dal. Components are flexible: the same boiled chana becomes chole one day and chaat another.

**Layer 2 — Weekday cooking (25-45 min each morning):** dishes that *consume* the components. Heat base + add boiled legumes = gravy in 10 minutes. Fresh tadka on pre-boiled dal. Rice always cooked fresh (it's the worst prep candidate and the easiest fresh cook).

Every recipe on this site states which layer it belongs to (**Type:** weekend-prep or weekday-dish) and which components it uses.

## The three non-negotiables

1. **Cook → cool to warm (30-40 min, never hours) → lid → fridge.** Morning food never waits on the counter till evening.
2. **Reheat thoroughly** — piping hot, not lukewarm. Reheat only the portion you're eating.
3. **Smell/look test beats any table.** Monsoon humidity shortens everything by a day if the fridge is opened often. When in doubt, throw it out.

## Daily flow

- **Morning (25-45 min):** cook the day's dish → cool while getting ready → fridge before work
- **Lunch:** reheat portion 1
- **Dinner:** reheat portion 2 (microwave: 2-3 min in the same glass box)
- **Night (5 min):** move tomorrow's freezer portion to fridge · scrape scraps to bin · rinse utensils · boiling water down the sink every other night

## Weekend prep sequence (everything overlaps)

1. Put legume round 1 in the pressure cooker
2. Start masala base in the kadai (stir between tasks)
3. Swap cooker batches as whistles finish (legume 2 → dal → potatoes)
4. Chop & box veggies while the base cooks down; knead/cook theplas if the week's plan uses them
5. Cool everything with lids ajar → label → fridge/freezer
6. Night: set curd, soak legumes/sprouts for the coming days

## Hygiene

Instahelp every 2-3 days for utensils + mopping. Pre-rinse everything so nothing cakes on; flex the schedule by nose in peak humidity.
